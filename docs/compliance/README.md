# GDPR Minimum Package (Commercializzazione)

Questo pacchetto copre il minimo operativo/documentale per passare da uso interno a prodotto vendibile.

## Contenuto

- `gdpr-informativa-template.md`: template informativa privacy (art. 13/14 GDPR)
- `registro-trattamenti-template.md`: struttura registro trattamenti
- `nomina-responsabili-template.md`: checklist nomine art. 28 GDPR
- `data-breach-procedura.md`: procedura gestione data breach
- `dsar-procedura.md`: procedura richieste diritti interessati + API disponibili

## Come usarlo

1. Compila i template con i dati reali della tua organizzazione.
2. Salva versione firmata/data di ciascun documento.
3. Definisci owner interno per ogni documento.
4. Riesamina trimestralmente (o dopo ogni cambio rilevante).

## Nota

Questi documenti sono template operativi tecnici. Prima del go-live commerciale, valida con consulente legale/privacy.
